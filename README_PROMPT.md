# Documentacion del Requerimiento Original

Este archivo contiene el prompt tecnico utilizado para evaluar la capacidad de respuesta y estructuracion de diferentes modelos de Inteligencia Artificial.

## Instrucciones de Desarrollo
"Actua como un desarrollador experto en HTML5 y accesibilidad. Requiero que crees un clon de la interfaz de YouTube cumpliendo estrictamente con las siguientes especificaciones:

1. Tecnologia: Utiliza unica y exclusivamente HTML semantico. Esta prohibido el uso de CSS y de JavaScript.
2. Estructura: El documento debe usar de forma correcta las etiquetas &lt;header&gt;, &lt;main&gt;, &lt;nav&gt;, &lt;aside&gt;, &lt;section&gt;, &lt;article&gt; y &lt;footer&gt;.
3. Configuracion Tecnica: Incluye en el &lt;head&gt; la codificacion charset="UTF-8", el viewport para adaptabilidad basica y una etiqueta &lt;meta&gt; description.
4. Layout: Ante la falta de CSS, utiliza tablas (&lt;table&gt;) para organizar la pagina en columnas (menu lateral, contenido principal y videos recomendados).
5. Contenido y Enlaces: Integra mediante etiquetas &lt;iframe&gt; los siguientes tres videos:
    - Video 1: https://www.youtube-nocookie.com/embed/kUMe1FH4CHE?start=1223
    - Video 2: https://www.youtube-nocookie.com/embed/a_iQb1lnAEQ
    - Video 3: https://www.youtube-nocookie.com/embed/SR5GxoFhIAU
6. Funcionalidad: Implementa un buscador funcional usando un &lt;form&gt; con un &lt;datalist&gt; que contenga los titulos de estos videos como sugerencias.
7. Documentacion: El codigo debe estar totalmente comentado usando etiquetas &lt;!-- --&gt; explicando la funcion de cada etiqueta.
8. Objetivo Final: Realizar el mejor esfuerzo para que la estructura sea ordenada, limpia y se parezca visualmente a YouTube usando solo herramientas nativas de HTML."
